# 🛠️ Installation

### Clone the repository

```bash
git clone https://github.com/unnaticreates/-Decodelabs-Internship.git
```

### Navigate to the project folder

```bash
cd your-repository-name
```

### Install Flask

```bash
pip install flask
```

### Run any project

Task 1

```bash
python project1/taskmanager.py
```

Task 2

```bash
python project2/expensetracker.py
```

Task 3

```bash
python pro3/password_generator.py
```

Task 4

```bash
python project4/app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
